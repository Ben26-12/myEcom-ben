Technical Highlights:
**Dynamic Routing**: Utilized React Router to manage navigation flows.
**Advanced Logic & Filtering**: Implemented real-time Product Search with Custom Hooks (like useDebounce) to optimize search performance and Price Filtering (Low to High, latest, etc) to enhance user discovery.
**Performance Optimization:** Implemented Lazy Loading and Code Splitting to significantly reduce initial page load time and improve user experience.
**Scalable UI Architecture:** Built a library of Reusable Components combined with SASS (SCSS) using a modular structure to ensure UI consistency and easy maintenance.
**Data & Form Handling:** Organized complex form logic and schema validation using Formik, Yup, and integrated Axios for API communication.
State & UI Management: Leveraged Context API for global state management and React Loading Skeleton for smooth data-fetching transitions.
